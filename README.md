# ☕ Core Java Practice & Revision

A comprehensive repository containing my **Core Java practice programs** and **hands-on coding exercises**. This repository has been built while learning and revising Java fundamentals, Object-Oriented Programming (OOP), Collections Framework, Exception Handling, Java 8 features, Multithreading, and various interview-oriented concepts.

The primary goal of this repository is to strengthen Java fundamentals through practical coding examples and prepare for Software Development and QA Automation interviews.
